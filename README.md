# bod_compose
# 一些 word 、ID 里用的 小而方便 的程序
- 藏文转换unicode后处理：།། ག། །།།། 等的空格问题。
- 正文样式问题处理：word 的正文样式出现藏文的话，这个文件在粘贴其它文档的时候就会出现乱码。就可以用这个VBA来处理。
- ID里藏文数字和阿拉伯数字切换的时候，可以直接把需要处理的区域反白。反白区里的数字可以改来改去了。
- 画法本框可以一次画满各式的框，可以画不同大小，粗细的框。十分方便。
